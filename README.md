# Daka

> 个人打卡记录查询工具

## Installation

-   needed：

    -   system：windows/*nix

    -   python version：python2.7

-   install requirements：

        pip install -r requirements.txt

## Usage

two ways to run：

1.  run directly：

        python daka.py

2.  generate a executable file to run：

        pyinstaller daka.spec
